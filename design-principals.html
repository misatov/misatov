Design Principals

Rule of Thirds

Shiseido

https://www.shiseido.com/us/en/

Shiseido utilizes Rule of Thirds in the unique and smart way. If you pay attention to where the price is, there is in the right-left point of the Rule of the Third.
In the other side of the point which is the left bottom is the key of the page which is the brief message about the value and "Shop Now" button.

PARC: Repetition

Oreo

https://www.oreo.com/

The main page graphic design is not only full of Oreo repetition that draw an eye to the viewers but the repetition of the colors that used in the OREO iD logo in the edges of the Oreos.
"Shop Now" button is set as the same color as the back ground color of the graphic images that is repeated.

PARC: Alignment

Flow Honey

https://www.honeyflow.com/

Alignment is applied in the main part o the page. You can easily find that the buttons are alined horizontally.
However when you can pay more close attention to the design, the start of the button of "Watch Video" and the end of the "Buy Now" button is vertically aligned withe paragraph.
Also, heading part and start and end of button types vertically.
